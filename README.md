**Inzopay** is a professional landing page designed for an electronic payment services platform that combines modern technology with the rich heritage of ancient Egyptian design. The page showcases payment solutions for businesses, online stores, and individuals, featuring a unique visual identity inspired by Egyptian pharaohs, gold accents, and desert tones. It includes sections for services, features, statistics, testimonials, pricing, contact information, and calls-to-action designed to convert visitors into users.

---

## ✨ Key Features

- 🏛️ **Ancient Egyptian Design Theme** – Royal navy, pharaoh gold, and sand tones create a distinctive brand identity.
- 📱 **Fully Responsive** – Works seamlessly on desktops, tablets, and mobile devices.
- 🎯 **Conversion-Optimized Layout** – Clear CTAs, hero section, and strategic placement of trust signals.
- 📊 **Statistics Display** – Shows platform metrics (40,000+ users, 160,000+ products).
- 🃏 **Feature Cards** – Six core benefits with SVG icons and hover animations.
- ⭐ **Testimonials** – Social proof section with star ratings and user avatars.
- 🗺️ **Service Breakdown** – Detailed explanation of how the platform works.
- 📞 **Contact Section** – Multiple contact methods (email, phone, live chat) with animated icons.
- 🔗 **External Links** – Directs users to the live platform for registration.
- 🎨 **Custom SVG Icons** – All icons are hand-coded SVG for sharp rendering on any screen.

---

## 🛠️ Technologies Used

- **HTML5** – Semantic structure with ARIA labels and SEO-friendly meta tags.
- **CSS3** – Custom properties (`:root`), Flexbox, CSS Grid, gradients, animations (`@keyframes`), and responsive media queries.
- **JavaScript (Vanilla)** – Smooth scroll-to-section functionality.
- **Google Fonts** – `Inter` font family for clean, modern typography.
- **SVG Icons** – All icons are inline SVG for pixel-perfect quality.

---

## 🚀 How to Use

1. **Clone or Download** this repository.
2. **Open `index.html`** in any modern web browser.
3. **Explore the page** – Navigate through sections using the header links or scroll naturally.
4. **Click the CTAs** – The "Try FREE for 14 days" button redirects to `https://inzopay.com/auth/sginup` (external).
5. **Customize** – Modify the content, colors, or images to fit your brand.
